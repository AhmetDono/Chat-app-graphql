# Chat App GraphQL

## Description
This project is a simple chat application developed to better understand the GraphQL subscription structure and enable real-time messaging. The application is built using React and Node.js, with GraphQL and MongoDB for data management.


## Açıklama
Bu proje, graphql subscription yapısını daha iyi anlamam için geliştirdiğim gerçek zamanlı olarak sohbet yapabilmesini sağlayan basit bir chat uygulamasıdır. Uygulama, React ve Node.js ile geliştirilmiştir ve veri yönetimi için GraphQL ve MongoDB kullanmaktadır.
